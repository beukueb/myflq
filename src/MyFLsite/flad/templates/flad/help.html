Programmatory access

Retrieve sequence with a FLADid

https://forensic.ugent.be/flad/sequence/xml/{FLADid}

Example: https://forensic.ugent.be/flad/sequence/xml/FA001

Retrieve FLADid with a sequence

https://forensic.ugent.be/flad/getid/xml/{sequence}

Example: https://forensic.ugent.be/flad/getid/xml/ACTGACTAGACATGCGAC

Retrieve FLADid with a sequence (add sequence to database if not present already)

https://forensic.ugent.be/flad/getid/xml/{sequence}?user={username}&password={password}

Example: https://forensic.ugent.be/flad/getid/xml/ACTGACTAGACATGCGAC?user=jeffreys&password=1984

Plain text instead of xml

To get a simple byte string back consisting only of the requested information, use 'plain' instead of 'xml'. In this case, when providing a FLADid to retrieve a sequence, only the sequence present in the database will be send back and not its complementary strand

Example: https://forensic.ugent.be/flad/getid/plain/ACTGACTAGACATGCGAC?user=jeffreys&password=1984
