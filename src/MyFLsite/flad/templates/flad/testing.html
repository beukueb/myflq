FLAX: testing FLAD

On this page, and with the corresponding API where you only have to replace 'flad' with 'flax', you can submit allele sequences for testing and development.
FLAX works exactly as FLAD, except that it is not possible to remove sequences from the database, and that the references are not guaranteed to be permanent. FLAX references start with 'FX' instead of 'FA'.
