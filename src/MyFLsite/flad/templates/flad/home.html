Mission statement

FLAD is a straightforward database for forensic alleles, that offers you permanent references to any expected allele sequence. If a sequence is not yet in the database, a simple registration is sufficient to start adding alleles to FLAD.
FLAD does not intend to get in the way of the forensic analyst and the interpretation of a forensic profile. It does not provide information about the loci a sequence belongs to, not even if it is a validated allele.

Database download

A weekly backup is made from the database that is available for download. It is encrypted with the forensic.ugent.be certificate, so that its origin can be determined for quality control.

Programmatory access

Retrieve sequence with a FLADid

https://forensic.ugent.be/flad/sequence/xml/{FLADid}

Example: https://forensic.ugent.be/flad/sequence/xml/FA001

Retrieve FLADid with a sequence

https://forensic.ugent.be/flad/getid/xml/{sequence}

Example: https://forensic.ugent.be/flad/getid/xml/ACTGACTAGACATGCGAC

Retrieve FLADid with a sequence (add sequence to database if not present already)

https://forensic.ugent.be/flad/getid/xml/{sequence}?u={username}&p={password}

Example: https://forensic.ugent.be/flad/getid/xml/ACTGACTAGACATGCGAC?u=jeffreys&p=1984
