Input File Requirements

#Locus name Allele number Allele Frequency Observed?
vWA 11
 0.2 1
vWA 12 0.2 x
vWA 13 0.6
FGA 13 0.3 xx
FGA 14 0.2 o
FGA 15 0.5

The input file has to be a comma-separated values text file (csv file). So the information in the table above should be formatted like:

#Locus name,Allele number,Allele Frequency,Observed?
vWA,11,0.2,1
vWA,12,0.2,x
vWA,13,0.6,
FGA,13,0.3,1
FGA,14,0.2,x
FGA,15,0.5,

  • The header row is not necessary. If one is used in the file, make sure that it starts with a "#"
  • All observed AND non-nobserved alleles and their corresponding fequencies need to be in the file
  • The locus name can be anything
  • The allele number can be anything
  • The allele frequency has to be a number between 0 an 1
  • Any character can be used to mark the allele as observed
