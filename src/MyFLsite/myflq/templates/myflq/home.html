MyFLq

Workflow

  1. Configure general (loci/alleles) specifications in Setup
    • Make a different configuration if the analysis needs different loci/alleles from previous analyses
    • Upload a file in the correct format for the loci configuration
    • Add the allele databank for the analysis by uploading a file in the correct format
    • submit the settings
  2. Choose a file to analyze and appropriate parameters in Analysis
    • After submitting the analysis request, it will show its progress state
  3. Retrieve the outcome in Results
    • Select the analysis for which you want to visualize results

BaseSpace

BaseSpace users that want to use forensic.UGent.be for custom loci and alleles csv file, need to make a configuration in Setup as outlined above.
When the configuration is uploaded, you can right-click and copy the url-address respectively for the loci file and the allele file, and paste it in the appropriate textbox on the BaseSpace MyFLq analysis launch page.
