MyFLq

Workflow

  1. Configure general (loci/alleles) specifications in Setup
    • Make a database if the analysis needs different loci settings from previous analyses
    • Add the primers either manually or by uploading a file in the correct format
    • Add the allele databank for the analysis by uploading a file in the correct format
    • Commit the settings for the database you want to use
  2. Choose a file to analyze and appropriate parameters in Analysis
    • After submitting the analysis request, it will show its progress state
  3. Retrieve the outcome in Results
    • Select the analysis for which you want to visualize results
