MyFLq source files

Webapp and commandline program

MyFLqApp.tar.gz

The archive contains all files to run this appication locally. In the top directory are the commandline programs that are also used by the web app. To run the web app, be sure to install the dependencies mentioned in the README.

Articles

MyFLq paper: http://dx.doi.org/10.1016/j.fsigen.2013.10.012
Pre-MyFLq STR paper: http://dx.doi.org/10.1016/j.fsigen.2012.03.004

Example files

Primers This file contains the primers for 16 commonly used loci. If you require the same set you can use it without altering, otherwise it can serve as an example of how the file should look.
Alleles This file contains the alleles that were present in the individuals of the samples used for the MyFLq paper. It can serve as an example to build your own allele databank file for use with MyFLq.
