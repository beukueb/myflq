MyFLq source files

Webapp and commandline program

MyFLqApp.zip

The archive contains all files to run this application locally. In the top directory are the commandline programs that are also used by the web app. To run the web app, be sure to install the dependencies mentioned in the README.

Example files

Loci This file contains the primers for 16 commonly used loci. If you require the same set you can use it without altering, otherwise it can serve as an example of how the file should look.
Alleles This file contains the alleles that were present in the individuals of the samples used for the MyFLq paper. It can serve as an example to build your own allele databank file for use with MyFLq.
